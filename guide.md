# MERIT Chat Bot

1. Install Ollama on your system by downloading it from the official website or using your package manager.

2. Install Python dependencies by running this command in your terminal:
```
pip install -r requirements.txt
```

1. Start the Ollama service by running:
```
ollama serve
```

2. Pull the tinyllama model:
```
ollama pull tinyllama
```

3. Run the chatbot:
```
python main.py
```
